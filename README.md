# MobileDev_TaskManagerApp

**Made by PG29 Julian R**
**Last Updated: 12/19/2025**

## Description
The project displays a simple Task Manager App with no persistence that holds functionality to: Add / Remove Groups, Add / Edit / Remove Tasks

### Controls
* To ADD a Group: Tap on + button
* To REMOVE a Group: Long Tap on any Group
* To CHECK a Group: Tap on any group

* To ADD a Task: Tap on + button
* To EDIT / REMOVE a Task: Long Tap on any Task
* To set a Task as COMPLETED: Tap on any Task
